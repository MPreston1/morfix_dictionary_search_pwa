Used only to obtain an anonymous auth token, the same way Morfix's own frontend does. Not a secret — this key just identifies the Firebase project.
Search tries a direct request first, then falls back through a chain of public CORS proxies (allorigins, codetabs, corsproxy.io). Only Authorization can be forwarded this way — browsers never allow JS to set Referer/Origin on any fetch, through any proxy. If Morfix's backend also enforces those, every attempt will fail with a 403 and you'll need a proxy you control that hardcodes those headers server-side.
