Used only to obtain an anonymous auth token, the same way Morfix's own frontend does. Not a secret — this key just identifies the Firebase project. Pealim needs no key at all — it's a plain page fetch.
Both dictionaries search through a chain of CORS routes: corsproxy.io first, then a direct request, then allorigins and codetabs as fallbacks. Morfix additionally requires its bearer token to be forwarded, which only corsproxy.io and the direct attempt can carry — so if corsproxy.io ever goes down or gets rate-limited, Morfix searches may fail even while Pealim's still work fine.
